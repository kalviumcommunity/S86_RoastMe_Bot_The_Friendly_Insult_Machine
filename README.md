# RoastMe Bot – The Friendly Insult Machine

## Project Overview

Ever felt like you needed a reality check? RoastMe Bot is your personal AI-powered insult generator, designed to keep you humble (or completely annihilate your ego). Whether you're looking for a lighthearted joke, a witty jab, or a soul-crushing roast, this bot has you covered.

With multiple roast levels and a comeback generator, it’s all fun and games—unless you go too far. But don’t worry, there’s an Apology Button if things get a little too spicy.

## Key Features

- Roast Levels – Choose from:

- Mild Burn (playful and lighthearted)

- Medium Roast (witty and slightly harsh)

- Instant Regret (savage, no holding back)

- AI-Powered Insults – Uses machine-generated phrases for custom, hilarious burns.

- Comeback Generator – Got roasted? Fire back with an equally brutal response.

- Friend Roast Mode – Enter a friend’s name, and let the bot do the dirty work.

- Apology Button – If the roast cuts too deep, hit this to generate an instant apology.
